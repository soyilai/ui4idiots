# UI for Idiots

A practical guide to UI design that doesn't assume you know what you're doing.

You've stared at a blank screen. You've opened Figma and closed it three times. Maybe you've grabbed a UI kit just to have something, anything on the canvas. Sound familiar? This course is built for that moment.

## What's Inside

Nine modules that take you from blank-canvas paralysis to actually shipping interfaces people can use.

- **Getting Started** — Break through the blank screen with three questions that unlock design intent
- **Structure & Hierarchy** — Control where the eye goes; learn why one primary element changes everything
- **Layout & Spacing** — Whitespace isn't wasted space. A spacing system replaces guesswork
- **Typography** — Five factors that separate readable text from eye strain
- **Color** — Build a palette deliberately; move beyond "it looks nice"
- **Depth & Layers** — Shadows and elevation create logic. Commit to one visual direction
- **Images & Media** — Aspect ratios, text over images, and designing for content you don't control
- **Polish** — Empty states, hover states, and the micro-interactions nobody brags about until they're missing
- **Real-World Transfer** — The same hierarchy and spacing that work on screens work on posters, signs, and physical objects

## Getting Started Locally

```bash
npm install
npm run dev
```

The site runs on `http://localhost:5173`

## Building

```bash
npm run build
```

Built with SvelteKit and Tailwind. Lessons are written in mdsvex.
